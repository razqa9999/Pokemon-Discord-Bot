# Pokemon-Discord-Bot
Pokemon Discord Bot adalah bot interaktif untuk menangkap, mengelola, dan melihat detail koleksi Pokémon secara langsung di server Discord Anda.

⚡ Pokemon Discord Bot 🎮Bawa keseruan petualangan Pokémon langsung ke server Discord Anda! Bot interaktif ini siap membantu Anda menangkap, mengelola, dan memamerkan koleksi Pokémon impian secara instan! ✨
